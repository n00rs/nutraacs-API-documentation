nuTRAACS API Documentation — Refund Cancel Ticket

Instructions to query refund or cancellation details for a ticket using the NutraACS API.

1. Overview

This document explains how to use the Refund Cancel Ticket API in the NutraACS system. The API allows users to query refund or cancellation details for a specific ticket.

2. API Endpoint

PUT https://nutraacs.com/api/voucher/ticket/refund_cancel_ticket_query

3. Headers

X-access-token: Bearer <access_token>

Replace <access_token> with a valid authentication token.

4. Request Payload

{
  "strTicketNumber": "8478486541",
  "strTicketSource": "THIRD_PARTY"
}
FieldTypeDescription
strTicketNumber string The ticket number for which refund or cancellation details are requested.
strTicketSource string Identifies the source system making the request (for example: THIRD_PARTY).

5. Example Request

curl -X PUT \
  https://staging-nutraacs.traacs.co/api/voucher/ticket/refund_cancel_ticket_query \
  -H "Content-Type: application/json" \
  -H "X-access-token: Bearer <access_token>" \
  -d '{
    "strTicketNumber": "8478486541",
    "strTicketSource": "THIRD_PARTY"
  }'

6. Notes

  • Ensure that the strTicketNumber exists in the system before querying.
  • A valid access token is required; otherwise, the request will be rejected.

7. Response

The API returns refund or cancellation information associated with the provided ticket number. The response structure may vary depending on ticket type, status, and workflow.
