NutraACS Integration Guide – Get Customers API

This API allows Ethiopian Airlines to retrieve all customer codes linked to their airline integration within NutraACS.

1. API Endpoint

POST /api/tenancy/tenancy/get_customers

3. Request Details

FieldTypeDescription
strVendorCodestringThe unique vendor code assigned to the integrated partner (e.g., airline, GDS, or other third-party system).

2. Example Request

{
  "strVendorCode": "vendorcode"
}

3. Example Responses

If the request is valid and customers are linked with the vendor, the API returns a list of customer codes and names.

✅ Success

[
  {
    "strCustomerCode": "IN0098",
    "strCustomerName": "TripTastic"
  },
  {
    "strCustomerCode": "IN0178",
    "strCustomerName": "Nomadic Adventure Co"
  }
]
